<file_contracts>
Standard file formats for subtask-to-subtask communication:

<structure_files>
Format: .extxyz (Extended XYZ)
Use for: Atomic structures with metadata (periodic and non-periodic systems)
Contains: Atomic positions, elements, periodic boundary conditions, lattice vectors

Reading with ASE:
```python
from ase.io import read
atoms = read("structure.extxyz")
# Access: atoms.positions, atoms.get_chemical_symbols(), atoms.cell, atoms.pbc
```

Writing with ASE:
```python
from ase.io import write
write("output.extxyz", atoms, format="extxyz")
```
</structure_files>

<tabular_data>
Format: .h5 (HDF5)
Use for: Tabular data with metadata

Structure:
```
data.h5
├── (root attributes): agent, created_at, note
├── runs/
│   ├── run_0/
│   │   ├── (attributes): run_note, topology_file, trajectory_file, ...
│   │   ├── step (dataset)
│   │   │   └── (attributes): units, description
│   │   └── temperature (dataset)
│   │       └── (attributes): units, description
```

Reading:
```python
import h5py

with h5py.File("../01_Task/data.h5", "r") as f:
    print(f.attrs.get("note"))
    run = f["runs/run_0"]
    data = run["temperature"][:]
    units = run["temperature"].attrs["units"]
```
</tabular_data>
