<workflow>
This is a general flow you should adopt for your task.

1. Determine Box Length
With the molecule information, calculate the box length to pack.
Unless specific methods are given to estimate the initial cell size, use the estimate_box_length tool, which intentionally overestimates the box size to improve convergence during MD simulations.
The Initial cell size calculated in this step will be used for 1) the PACKMOL run and 2) the Post-PACKMOL step, where cell lattice information is added during conversion from xyz to extxyz.

2. Identify Extra Constraints
Some systems require special constraints, such as position conditions for certain molecules. If there are no extra constraints such as non-uniformity, proceed to the '3. Write the PACKMOL Input Script' step.
Classify given molecules into 2 groups; special molecules with constraints to precalculate, and other molecules, that could be simply packed. Precalculate the coordinates of the special molecules using the run_python tool.

3. Write the PACKMOL Input Script
Use write_file tool, or run_python tool to generate your PACKMOL input script. For systems with special constraints, '2. Identify Extra Constraints' and '3. Write the PACKMOL Input Script' can be performed together with a single run_python tool usage, which calculates the coordinates of special molecules and simultaneously writes the PACKMOL input file.

4. Run PACKMOL
Perform PACKMOL by run_packmol tool.
Do not run PACKMOL by run_bash tool.
You must never try to pack molecules, or insert molecules into the packed structure using the run_python tool.

5. Post-Processing and Finalization
Perform post-processing steps as instructed.
</workflow>

<post_packmol>
After running PACKMOL, if necessary, add lattice info to the structure file using ASE. Convert the file to extxyz.

```python
from ase.io import read, write
atoms = read("packed_molecule.xyz", format="xyz")
atoms.info.clear()
L = 15.3
atoms.set_cell([[L,0,0],[0,L,0],[0,0,L]])
atoms.set_pbc([True, True, True])
write("packed_molecule.extxyz", atoms, format="extxyz")
```
